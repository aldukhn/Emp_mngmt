### _**WARNING: This project may not work with you (Only Test)**_

## Installation

```bash
$ composer update
```
then crate new tables with:

```bash
$ php artisan migrate
```
then create default system user

```bash
$ php artisan db:seed
```

```bash
$ php artisan key:generate
```

```bash
$ php artisan config:clear
```
then start it with
```bash
$ php artisan serve
```
